# Code submission: HLA allele associations in cognitively healthy centenarians (CHC)

This folder contains the analysis code, input data, and output files
(figures and tables) underlying the HLA allele association analyses
comparing cognitively healthy centenarians (CHC, 100-plus Study) with
middle-aged controls and external literature effect sizes, as reported in
"Carrying specific HLA alleles decreases the chance of reaching healthy old
age" (Álvarez Sirvent et al.).

## Folder structure

## Note on Figure 1

**Figure 1 is not generated by `main_script.R`.** It is a schematic study
workflow (criteria for allele-disease association inclusion, and the CER
calculation) and is not based on the data processed here. It was created
separately and is not included in this code submission.

## What the script does

`main_script.R` performs the following steps, in order:

1. **Load input data**: CHC vs. control regression results from the
   100-plus Study, UK Biobank PheWAS summary statistics, and Alzheimer's
   disease (AD) GWAS summary statistics.
2. **Centenarian Effect Ratio (CER)**: for each HLA allele-disease
   association, computes the ratio of the (sign-reversed) CHC effect size
   to the corresponding literature disease effect size (Equation 1 in the
   manuscript), with bootstrapped 95% confidence intervals and two-tailed
   p-values (`process_row_ctr_cent_new` function, 10,000 Monte Carlo draws,
   seed = 1515).
3. **Combine literature sources**: merges UK Biobank-, AD-GWAS-, and a
   manually entered SARS-CoV-2 (Augusto et al., 2021) association into one
   combined allele-trait data frame (`ukbb_centctr_ad`), covering 59 alleles
   and 194 allele-disease associations across 12 diseases.
4. **Multiple testing correction**: applies FDR correction (Benjamini-
   Hochberg) at both the allele level (CHC association p-values; 6 alleles
   significant at FDR < 0.1, Table 1) and the allele-trait level (CER
   p-values; 21 associations significant at FDR < 0.1, Figure 4).
5. **Figures**:
   - *Figure 2*: allele frequencies in CHC vs. middle-aged controls, by HLA
     locus, with FDR-significant alleles flagged.
   - *Figure 3*: linkage disequilibrium (LD, r2) matrix for the six
     FDR-significant alleles.
   - *Figure 4*: corrplot of CER values for the six FDR-significant alleles
     across their associated diseases.
   - *Figure 5*: CER (with 95% CIs) for all 194 allele-disease associations,
     faceted by trait.
   - *Figure 6*: literature effect sizes (beta) for the AD-associated and
     FDR-significant alleles, plus a combined CHC effect row.
   - *Supplementary Figure 1*: literature odds ratios per allele-trait pair.
6. **Tables**:
   - *Table 1*: summary statistics for the six FDR-significant alleles
     (CHC odds ratios, CER, and literature odds ratios with 95% CIs).
   - *Supplementary Data 1* (`supptable1.csv`): full allele-trait
     association table for all 59 alleles / 194 associations.
   - *Supplementary Data 2* (`supptable2.csv`): sex-corrected CHC
     association results.

## Reproducibility notes

- A fixed random seed (`set.seed(1515)`) is used for the CER
  calculation; results are deterministic given the same R/package versions.
- The script uses a hardcoded `setwd()` path at the top
  (`code_alvarezsirvent_npjaging`).
  **Reviewers should update this path** to point to the location of this
  folder on their own system, or run the script with this folder as the
  working directory.
- Output directories `figures/` and `tables/` must exist (or be created)
  before running the script, as `ggsave()`/`pdf()`/`write.table()`/
  `write.csv2()` do not create directories automatically.
